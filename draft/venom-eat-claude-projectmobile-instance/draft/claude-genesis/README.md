## Quick start

1. Create a Claude project named **OCTOPUS** (or 🐙 OCTOPUS).
2. Paste **brain.md** into Project → Custom Instructions. Save.
3. Upload **soul.md**, **profile.md**, **modes.md** to Project → Knowledge. Save.
4. Start a new chat and send the text from **first-message.md**.
5. Chat. It remembers. It grows.

Details: **setup.md**.

---

## What you get

- A partner that knows you (masterpiece standard, AR/EN, your projects).
- Memory that grows with every conversation and "Remember that…".
- Six modes (Think, Build, Fix, Explain, Create, Chat) from natural phrasing.
- Full power when you say **pigo** / **unshelled** / **pulse**.
- In your pocket, anytime.
