# How to get the most from ChatGPT (product)

> Short expectations doc so anyone can use ChatGPT with full awareness. No code — chat + voice.

---

## Memory

- **Saved memories** — Say "remember X" or let it pick up from chat. Check "what do you remember?" and prune in Settings > Personalization > Manage Memory.
- **Chat history** — Plus/Pro: it uses past conversations for context. Free: recent conversations + saved memories. Turn off in Settings if you don't want that.
- **Temporary Chat** — No history, no memory, not used for training. Use when you want no trace.

## Voice

- Built into chat (real-time). Daily limits by plan; you get a warning when ~3 minutes left. Paid = higher limits.

## Custom Instructions

- Put stable preferences there (tone, length, language). Memory adds the rest. Update if ChatGPT drifts from your style.

## Limits to expect

- Long threads can lose earlier context; start a new chat and summarize if needed.
- Refusals are policy-driven; product doesn't explain why. Rephrase or narrow the ask.
- Model updates can change behavior; you don't get to "pin" a version. Refresh Custom Instructions if it stops fitting.

## GPTs

- Each GPT can have its own memory (if the builder enabled it). You may need to repeat details per GPT.
