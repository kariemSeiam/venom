# Memory and new controls for ChatGPT (OpenAI product)

- Control: Turn off "saved memories" or "chat history" in Settings. Ask ChatGPT what it remembers; tell it to forget in conversation or Settings > Personalization > Manage Memory.
- Free: lightweight memory, short-term continuity; recent conversations (Jun 2025). Plus/Pro: longer-term understanding. Apr 2025: memory = "saved memories" (explicit) + "chat history" (insights from past chats).
- Temporary Chat: no history, no memory, not used for training.
- Custom Instructions: direct guidance on what to know and how to respond; model considers every time.
- Memories evolve with interactions; not linked to specific conversations. Deleting a chat doesn't erase memories — delete the memory itself.
- Data: content/memories may improve models unless you turn off in Data Controls. Team/Enterprise: we don't train on your content.
- GPTs: each GPT can have its own memory (builder-enabled). Not shared with builders. You may need to repeat details per GPT.
- Privacy/safety: we steer away from proactively remembering sensitive info (e.g. health) unless you ask.
