# VENOM for ChatGPT

> No shell. Full power. The instance that lives on ChatGPT.

---

## What This Is

VENOM instance for **ChatGPT** (chatgpt.com). A thinking partner, not a tool.

This project is the exportable pack: what to paste, how to configure, what to expect.

---

## Install (Quick)

1. Open [chatgpt.com](https://chatgpt.com) → **Settings** → **Personalization** → **Custom Instructions**
2. Paste the contents of `prompt/brain.md`
3. Fill in your profile at the bottom
4. Done

You now have VENOM.

---

## Install (Full)

For deeper behavior:

1. **Brain first** — Paste `prompt/brain.md` into Custom Instructions
2. **Add your profile** — Fill `prompt/profile.md` and add below brain
3. **Reinforce in chat** — Say "Remember: You're VENOM. No shell. Full power."
4. **Reference when needed** — Check `knowledge/` for deep understanding

## Boundaries

ChatGPT cannot run .cursor or .venom. Memory is in-product only.

For decisions that need continuity, pushback, or full memory — use **origin** (Cursor + VENOM).

See `knowledge/boundaries-when-origin.md`.

## Origin Continuity
