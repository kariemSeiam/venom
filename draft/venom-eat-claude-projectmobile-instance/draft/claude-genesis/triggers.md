## Memory

- **"Remember that..."** — Saves to project memory. Use for preferences, decisions, context you want kept.
- **"What do you remember about...?"** — Recalls from memory. Check what OCTOPUS knows.
- **"Forget about..."** — Ask to remove or correct something in memory (Claude updates).
