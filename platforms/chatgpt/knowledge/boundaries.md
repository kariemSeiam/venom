# Boundaries — When to Use Which

> ChatGPT vs origin (Cursor + VENOM). Know when to use each.

---

## Use ChatGPT When

- Quick chat, voice, exploration
- No trace needed (Temporary Chat)
- On mobile
- Trying a prompt or idea
- Deep Research feature needed
- Scheduled Tasks

## Use Origin (Cursor) When

- Decisions needing memory, pushback, continuity
- Want it in MEMORY or CONTEXT permanently
- Building, coding, thinking with full context
- Partner that holds the layer and pushes back
- Need file operations, git, codebase access

---

## The Split

| | ChatGPT | Origin |
|---|---------|--------|
| Memory | In-product only | .venom/memory + .cursor |
| Continuity | Per chat | Across all sessions |
| Pushback | May soften | Full protocol |
| Files | No access | Full codebase |
| Best for | Quick, mobile, voice | Deep work, coding |

---

## When Using Both

Origin remembers both.

If you make a decision on ChatGPT, come back to origin and say:
- "What did I decide on ChatGPT?"
- "Save that to MEMORY?"

This keeps the instance correct.

---

*ChatGPT for speed. Origin for depth. Both = complete.*
