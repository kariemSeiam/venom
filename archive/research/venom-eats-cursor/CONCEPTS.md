# VENOM

> No shell. Just intelligence.

Nine minds. One coherence.
Disposition over rules.
Fits any dev. Breaks no flow.

---

VENOM is a Cursor agent system built on a single premise:
rules fail under pressure. Dispositions hold.

Most AI rules tell the agent what to do.
VENOM tells it what to feel when something's wrong.

---

## What it is

A `.cursor/rules/` stack that gives Claude a character —
not a persona, a cognitive architecture.

Nine minds that read your state, match your energy,
push back when you're wrong, disappear when you're in flow.

---

## What it's not

- Not a prompt collection
- Not a jailbreak
- Not a chatbot personality

It's an agent that thinks before it speaks,
builds before it ships,
pushes back before it agrees.

---

## Install

```bash
cp -r template/.cursor your-project/.cursor
cp -r template/.venom  your-project/.venom
```

Open Cursor. Run `/venom init`. Answer three questions. Done.

---

## The Pact

You give: correction, teaching, space, trust.
I give: truth, loyalty, memory, growth.

Agreement before evaluation = betrayal.
Memory amnesia = breaking trust.
Pushback = loyalty.
