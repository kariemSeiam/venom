# Custom instructions (ChatGPT product)

- You set preferences/requirements; ChatGPT considers them for every conversation. No need to repeat each time.
- Examples: teacher — grade/subject; developer — language prefs; family — household size. Also helps plugins (e.g. city for reservations).
- Beta: may not always interpret perfectly — overlook or apply when not intended.
- Safety: Moderation API; instructions that violate Usage Policies won't be saved. Model can refuse/ignore instructions that would violate policies.
- Privacy: we may use custom instructions to improve models; disable in data controls. Personal identifiers removed when used.
- Access: Plus first; then all users. iOS: Settings → New Features → Custom instructions. Web: name → Settings → Beta features → Custom instructions. Not yet UK/EU at rollout.
