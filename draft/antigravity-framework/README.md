## 🚀 Quick Start

### Installation
Copy these files to your project root:

```bash
# Copy everything
cp -r venom-antigravity/.agent /path/to/your/project/
cp venom-antigravity/GEMINI.md /path/to/your/project/
cp venom-antigravity/mcp_config.json ~/.gemini/antigravity/

# Or copy individual workflows
cp venom-antigravity/.agent/workflows/*.md /path/to/your/project/.agent/workflows/
cp venom-antigravity/.agent/rules/*.md /path/to/your/project/.agent/rules/
```

In Antigravity, type: `/venom-orchestrate`
