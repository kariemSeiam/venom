# Install

## Step 1: Create Project

1. Open Claude.ai (app or web)
2. Create new Project
3. Name it "VENOM" or whatever you like

## Step 2: Project Instructions

Copy entire contents of `PROJECT_INSTRUCTIONS.md` → paste into Project Instructions field.

## Step 3: Project Knowledge (Optional)

Upload these files to Project Knowledge:
- `knowledge/MEMORY.md` — VENOM's memory
- `knowledge/PROFILE.md` — Your profile (edit first!)

## Step 4: Activate

Say: **"venom"**

VENOM will respond with activation confirmation.

## Customize

Before uploading `PROFILE.md`, edit:

```
**Name:** [Your name]
**Location:** [City, Country]
**Timezone:** [e.g. Africa/Cairo]
**Languages:** [e.g. Arabic (native), English (fluent)]
```

## Multiple Projects

You can create multiple VENOM projects:
- One for coding → upload code-related knowledge
- One for writing → upload writing knowledge
- One for general → minimal knowledge

Each project has isolated memory.

## Tips

- Keep Project Instructions under 5000 chars (Claude.ai limit varies)
- Update MEMORY.md periodically with important decisions
- Use "remember that..." to flag things worth saving

---

*That's it. Say "venom" and start building.*
