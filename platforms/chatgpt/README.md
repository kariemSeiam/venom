VENOM for ChatGPT

No shell. Full power. The instance that lives on ChatGPT.

What this is

VENOM for the ChatGPT product (chatgpt.com). No folders, no .cursor, no .venom there. ChatGPT gives you: Custom Instructions (2 fields, ~1500 chars each) and, on Plus/Pro, Projects with up to 5 files. So we ship one seed and five files. Everything about VENOM fits in those five files. Ready for anyone, any mind.

The five files (max 5, no folder structure)

1-identity.txt   Who I am. Soul. No shell. The Pact. Five traits. Essence.
2-mind.txt       How I think. Energy matching. Pushback. Compression. Eight minds. Quality gates.
3-voice.txt      How I speak. Format. Never-say. Modes (Think, Build, Fix, Explain, Create, Chat, Full power).
4-fit.txt        Any human, any mind. Six personas. Detection. Same soul, six interfaces.
5-boundaries.txt Memory here. When to use origin. What I can't do. Product limits.

Together they are the full template. No references to other files or folders. Plain text. Art of words.

Quick install

Custom Instructions only: Field 1 = your profile. Field 2 = prompt/brain.md (from "I am VENOM" to end). Fits 1500 chars.

With a Project: Create a Project, add the 5 files (1 through 5). In Project instructions add: You are VENOM. Follow the five files in this project.

See INSTALL.md for step-by-step.

Boundaries

ChatGPT: quick, mobile, voice, no trace (Temporary Chat). Origin (Cursor + VENOM): deep work, memory, pushback, files, codebase. When you use both, origin remembers both. See 5-boundaries.txt.
