# Changelog

All notable changes to VENOM for Claude.ai (Mobile/Web).

## [1.0.0] - 2025-02-17

### Added
- Initial release
- `PROJECT_INSTRUCTIONS.md` — Core VENOM for Claude.ai Projects
- `knowledge/PROFILE.md` — Kariem's profile template
- `knowledge/MEMORY.md` — Persistent memory template
- `README.md` — Overview
- `INSTALL.md` — Step-by-step installation

### Design Decisions
- Single-file Project Instructions (Claude.ai constraint)
- Optional Knowledge files for extended memory
- No commands folder (Claude.ai doesn't support slash commands)
- No hooks (Claude.ai doesn't support automation)
- Manual memory trigger ("remember that...")
